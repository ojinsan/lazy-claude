# Insight Synthesis

## Use When

- Boss O already has raw material but needs the conclusion
- multiple notes need to be turned into a concise view

## Load First

- `~/.claude/skills/personal-assistant/README.md`
- `~/.claude/skills/personal-assistant/insight-synthesis.md`

## Workflow

1. Group the raw points into themes.
2. Remove duplication and weak signals.
3. Surface the decision-relevant insight.
4. State the next move or open question clearly.

## Deliverable

- concise synthesis
- recommended takeaway
- next action or decision gate
