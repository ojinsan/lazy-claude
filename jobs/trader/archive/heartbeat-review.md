# Heartbeat Review

## Use When

- on trading day heartbeat checks
- after several short-window monitoring runs have accumulated local notes

## Load First

- `~/workspace/roles/trader.md`
- `~/.claude/skills/trader/realtime-monitoring.md`
- `~/.claude/skills/trader/thesis-evaluation.md`

## Workflow

1. Review recent local monitoring notes.
2. Compress them into the most interesting changes in behavior.
3. Identify whether the active trade plan is strengthening, weakening, or invalidated.
4. Save only the meaningful heartbeat conclusions to local storage.

## Deliverable

Produce a compact heartbeat summary with:
- interesting developments
- thesis drift check
- action relevance

## Safety

- Heartbeat is for signal, not noise.
- If nothing meaningful changed, say so plainly.


## Layer alignment

Layer 3 stock-overseeing support job for 30-minute heartbeat compression.
