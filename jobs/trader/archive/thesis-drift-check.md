# Thesis Drift Check

## Use When

- a live position needs review
- Boss O asks whether the original idea still holds
- monitoring suggests the market behavior has changed

## Load First

- `~/.claude/skills/trader/README.md`
- `~/.claude/skills/trader/thesis-evaluation.md`
- `~/.claude/skills/trader/orderbook-reading.md`
- `~/.claude/skills/trader/market-structure.md`
- `~/.claude/skills/trader/risk-rules.md`

## Workflow

1. Restate the original thesis and invalidation.
2. Compare it against current price behavior, flow, and structure.
3. Identify whether the thesis is intact, weakening, or broken.
4. Recommend hold, reduce, cut, or wait.

## Deliverable

Produce a drift note with:

- original thesis
- current evidence
- drift status
- recommended response

## Safety

- Never hide negative drift.
- If the thesis is broken, say it clearly.
