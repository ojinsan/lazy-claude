# Morning Brief

## Use When

- Boss O asks for a start-of-day summary
- insight gathering and deep analysis are complete

## Load First

- `skills/trader/CLAUDE.md`
- `playbooks/trader/archive/insight-gathering.md`
- `playbooks/trader/archive/deep-analysis.md`
- `~/.claude/skills/trader/trade-planning.md`
- `~/.claude/skills/trader/risk-rules.md`

## Workflow

1. Compress raw prep and deep analysis into a short decision-ready brief.
2. Prioritize only the names that matter now.
3. Make the action ladder explicit: watch, enter, reduce, avoid, or wait.
4. End with the main risks and unresolved questions.

## Deliverable

Brief structure:

- market tone
- priority tickers
- action plan
- risk notes
- what Boss O should watch at the open

## Safety

- Draft first.
- Only send it externally if Boss O has explicitly approved that delivery.
