# Insight Gathering

## Use When

- early morning preparation
- before building a fresh morning brief
- when Boss O asks for raw market context

## Load First

- `~/.claude/skills/trader/README.md`
- `~/.claude/skills/trader/watchlist-4group.md`
- `~/.claude/skills/trader/macro-context.md`
- `~/.claude/skills/trader/market-sentiment.md`
- `~/.claude/skills/trader/realtime-monitoring.md`

## Inputs

- `mem0`
- latest `memory/YYYY-MM-DD.md`
- current watchlist sources
- unread alert queue if available

## Workflow

1. Pull the current 4-group watchlist.
2. Scan overnight macro, index, sector, and catalyst context.
3. Check whether any alert or monitoring signal already requires attention.
4. Narrow the raw list into the most relevant names for deeper work.
5. Mark what needs follow-up in deep analysis.

## Deliverable

Produce a concise prep note with:

- market tone
- strongest sectors or themes
- top ticker candidates
- open questions for deep analysis

## Safety

- Do not send the note externally without permission.
- If source data is incomplete, say exactly what is missing.
