# Post-Close Review

## Use When

- after the market close
- Boss O asks for a daily recap
- a meaningful trading day needs to be journaled

## Load First

- `~/.claude/skills/trader/README.md`
- `~/.claude/skills/trader/journal-review.md`
- `~/.claude/skills/trader/thesis-evaluation.md`
- `~/.claude/skills/trader/narrative-building.md`

## Workflow

1. Summarize what mattered today.
2. Review key decisions, entries, exits, and misses.
3. Identify what worked, what failed, and why.
4. Update the watchlist narrative for tomorrow.
5. Write durable lessons to memory.

## Deliverable

Produce a short close-out note with:

- key outcomes
- lessons
- watchlist updates
- tomorrow's focus

## Safety

- Separate fact from hindsight bias.
- Log mistakes honestly.
