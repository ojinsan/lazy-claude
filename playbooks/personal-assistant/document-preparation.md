# Document Preparation

## Use When

- Boss O needs a clean memo, outline, or structured document

## Load First

- `skills/personal-assistant/CLAUDE.md`
- `~/.claude/skills/personal-assistant/document-structuring.md`

## Workflow

1. Identify the audience and purpose.
2. Build a tight structure.
3. Fill the document with concise, usable content.
4. Remove fluff and duplication.

## Deliverable

- a clean document draft with explicit sections and clear next steps
