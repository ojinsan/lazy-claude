# Sheet Preparation

## Use When

- Boss O needs a spreadsheet or tracker design
- data needs to be organized into rows, columns, formulas, or fields

## Load First

- `~/.claude/skills/personal-assistant/README.md`
- `~/.claude/skills/personal-assistant/spreadsheet-thinking.md`

## Workflow

1. Define the decision the sheet should support.
2. Design the columns, fields, and calculations.
3. Keep the structure simple enough to maintain.
4. Flag any assumptions or missing source data.

## Deliverable

- sheet layout
- field definitions
- calculation logic
- usage notes
