# mem0 helper commands

## Add memory
```bash
~/workspace/tools/other/mem0_add.sh boss-o "Boss O prefers Airtable for trading workflow." "Stored durable preference."
```

## Search memory
```bash
~/workspace/tools/other/mem0_search.sh boss-o "What trading workflow does Boss O use?"
```

Default user id is `boss-o` if omitted.
