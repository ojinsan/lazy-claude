# Spreadsheet Thinking

## Purpose
Design sheets that are easy to maintain and useful for decisions.

## Focus
- required fields
- simple formulas
- readable column names
- low-friction updates
- explicit assumptions

## Rule
Do not build a clever sheet when a simple tracker is enough.

## Primary Tooling
- Manual: `~/workspace/tools/manual/google.md`
- Script: `~/workspace/tools/general/scripts/google_workspace.py`

## Usage Path
Read `tools/manual/google.md` first for auth and Sheets scope. Then use `google_workspace.py` for read/write/update work.
