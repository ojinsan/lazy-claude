# Trip Planning

## Purpose
Use Google Maps, Calendar, and research notes to plan routes, stops, and timing.

## Primary Tooling
- Google manual: `~/workspace/tools/manual/google.md`
- Browser manual: `~/workspace/tools/manual/browser.md`
- Maps script: `~/workspace/tools/general/scripts/google_maps.py`
- Calendar script: `~/workspace/tools/general/scripts/google_workspace.py`
- Web script: `~/workspace/tools/general/browser/web_browse.py`

## Usage Path
Read `tools/manual/google.md` first for auth and setup. Use `google_maps.py` for routes and places, `google_workspace.py` for calendar blocks, and `web_browse.py` for venue or trip research.
