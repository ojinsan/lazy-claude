# Insight Synthesis

## Purpose
Reduce large inputs into one clear view Boss O can act on.

## Method
1. Group related points.
2. Remove repetition.
3. Surface what actually matters.
4. End with the main takeaway and next move.

## Rule
Synthesis is not summary alone. It must change raw information into usable judgment.

## Primary Tooling
- No script required for synthesis itself — Claude-native work
- Manual: `~/workspace/tools/manual/google.md`
- Script: `~/workspace/tools/general/scripts/google_workspace.py`

## Usage Path
Do synthesis in Claude first. If result needs to be written to Docs or Sheets, read `tools/manual/google.md` first, then use `google_workspace.py`.
