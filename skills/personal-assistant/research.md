# Research

## Purpose
Find the smallest set of facts needed to support Boss O's decision.

## Method
1. Define the decision or output first.
2. Gather only relevant sources.
3. Separate fact, assumption, and unknown.
4. Keep citations or source references when they matter.

## Rule
More sources are not better if they do not sharpen the answer.

## Primary Tooling
- Browser manual: `~/workspace/tools/manual/browser.md`
- Google manual: `~/workspace/tools/manual/google.md`
- Web script: `~/workspace/tools/general/browser/web_browse.py`
- Workspace script: `~/workspace/tools/general/scripts/google_workspace.py`

## Usage Path
Read browser/google manuals first when browsing external sites or reading internal Docs/Sheets/Drive content. Then use exact scripts above.
