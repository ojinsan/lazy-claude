# SID Tracker

## Purpose

Track recurring participant behavior and whether a move has real sponsorship.

## Watch For

- persistent names behind the move
- whether support continues across sessions
- whether the move is broad or concentrated

## Output

State whether the move looks:

- sponsored
- opportunistic
- fading
- unclear
## Tool Resolution

When a skill needs code or helper scripts, resolve them from `~/.claude/tools` using the matching role folder first, then `other` if needed.

Examples:
- trader skills -> `~/.claude/tools/trader`
- personal-assistant skills -> `~/.claude/tools/personal-assistance`
- content-creator skills -> `~/.claude/tools/content-creator`
- shared helpers -> `~/.claude/tools/other`
