# Technical Analysis

## Purpose

Read price, volume, structure, momentum, and technical positioning to decide whether a ticker deserves attention, patience, or avoidance.

## Focus

- market structure
- support / resistance
- trend quality
- volume confirmation
- BOS / CHoCH if available
- indicator context
- swing readiness

## Tool Resolution

Primary tools:
- `~/.claude/tools/trader/market_structure.py`
- `~/.claude/tools/trader/indicators.py`
- `~/.claude/tools/trader/wyckoff.py`
- `~/.claude/tools/trader/screener.py`

## Rule

Do not confuse movement with structure. Price going up is not enough. The setup needs context, confirmation, and a clean invalidation.
