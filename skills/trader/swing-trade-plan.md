# Swing Trade Plan

## Purpose

Create a basic swing plan using price, volume, structure, and narrative alignment.

## Focus

- setup type
- entry zone
- invalidation
- target / reduce area
- duration assumption
- what confirmation is still missing

## Tool Resolution

Primary tools:
- `~/.claude/tools/trader/tradeplan.py`
- `~/.claude/tools/trader/market_structure.py`
- `~/.claude/tools/trader/indicators.py`
- `~/.claude/tools/trader/api.py`

## Rule

Basic swing plan means structure first. If volume and price do not support the thesis, do not force the plan.
